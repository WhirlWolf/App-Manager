# App-Manager
App related features and automations such as enabling auto rotate when gallery is launched, etc.
</div>

---

## Prerequisites
- [Tasker](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm)

## Setup
- [Import Project](https://taskernet.com/shares/?user=AS35m8m8L9YzBV3qbzaAAqHiSYXYBbD3QfZ7hr0hRK4ojOFTCrjWh2CScbjMw4NaudRi1zKKzq85&id=Project%3AApp+Manager)
- Run setup task

## Features
- List apps
- Manage permissions
- List activities
- Copy apk
- Copy icon
- Configure automations per app
  - Force stop on boot
  - Force stop daily
  - Enable auto rotote on launch
  - Enable bluetooth on launch
  - Enable mobile data on launch
  - Enable wifi on launch
  - Enable airplane mode on launch
  - Enable location on launch
  - Enable battery saver on launch
  - Enable DND on launch
  - Enable auto brightness on launch
- See app in play store
- App details
- Install alert
- Uninstall alert
- Updated alert with new version
- App lock
